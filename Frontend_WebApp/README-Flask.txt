README: Installing Flask to Run Local Server for Web Application

Reference: http://flask.pocoo.org/docs/1.0/installation/

Open Terminal and Run
	pip install Flask

To run the local server:
	cd to adequate project folder
	export FLASK_APP=flaskr
	export FLASK_ENV=development
	flask run

On browser open the indicated local address: http://127.0.0.1:5000/ or similar
